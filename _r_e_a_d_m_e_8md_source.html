1 Kinetic-C Client Library
2 ========================
3 ![](http://travis-ci.org/atomicobject/kinetic-c.png?branch=master)
4 
5 This repo contains code for producing C Kinetic clients which use the Seagate Kinetic protocol. Code examples/utilities that use the Kinetic C library are included for reference and usage during development.
6 
7 [Kinetic-C build status](http://travis-ci.org/atomicobject/kinetic-c) is provided via [Travis CI](http://travis-ci.org)
8 
9 Getting Started
10 ================
11 
12  > git clone --recursive https://github.com/atomicobject/kinetic-c.git
13 
14  > cd kinetic-c
15  > bundle install # Ensures you have all RubyGems needed
16  > bundle execute rake # Run all tests and build kinetic-c library and examples

41 Common Developer Tasks
42 ======================
43 
44 NOTE: Prefix the following commands with `bundle exec` so that they execute in the context of the bundle environment setup via `bundle install`.
45 
46 * List available Rake tasks w/descriptions
47  * `rake -T`
48 * Run all tests and build the library and examples
49  * `rake`
50 * Build the library
51  * `rake release`
52 * Run all unit/integration tests
53  * `rake test:all`
54 * Analyze code
55  * `rake cppcheck`
56 * Generate documentation
57  * `rake doxygen:gen`
58 * Apply license to source files (skips already licensed files)
59  * `rake apply_license`
60 * Build/install Google Protocol Buffers support for the Kinetic-Protocol
61  * `rake proto`
62 * Enable verbose output (for current Rake run)
63  * `rake verbose *<task_A>* *<task_B>*`
